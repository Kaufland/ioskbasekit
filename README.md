# KLBaseKit

is a collection of iOS utilities and functions. It also includes **a full featured** REST client.

## Features

 - REST client with object mapper and certificate pinning
 - Logging
 - String obfuscation
 - Multicast delegates
 - Utilities for DispatchQueue & FileManager
